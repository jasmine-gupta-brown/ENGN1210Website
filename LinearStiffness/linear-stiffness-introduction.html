SPRINGS IN SERIES

Force Balance :-
\[F = F_1 = F_2\] Geometric Compatibility :-
\[\Delta X = \Delta X_1 + \Delta X_2\] Model :-
\[F_1 = k_1 \Delta X_1 \hspace{2em} and \hspace{2em} F_2 = k_2 \Delta X_2\]
Equivalent stiffness = \(k_{eff}\)
(effective)
\[F= k_{eff} \Delta X\] \[\begin{array}{ccc} \Rightarrow F= k_{eff} (\Delta X_1 + \Delta X_2)\\ \\ \Rightarrow F= k_{eff} \left( \dfrac{F_1}{k_1} + \dfrac{F_2}{k_2} \right) = k_{eff} F \left( \dfrac{1}{k_1} +\dfrac{1}{k_2} \right) \end{array}\]
\[\begin{array}{ccc} \Rightarrow \cancel{F} = k_{eff} \cancel F \left( \dfrac{1}{k_1} + \dfrac{1}{k_2} \right)\\ \\ \Rightarrow k_{eff} = \left( \dfrac{1}{k_1} + \dfrac{1}{k_2} \right)\\ \\ \text{Therefore, for spring in series:}\hspace{20em}\\ \\ \boxed{k_{eff} =\dfrac{1}{k_1} + \dfrac{1}{k_2} + ... + \dfrac{1}{k_N}}\\ \end{array}\]
SPRINGS IN PARALLEL

Force Balance :-
\[F = F_1 + F_2\] Geometric Compatibility :-
\[\Delta X = \Delta X_1 = \Delta X_2\] Model :-
\[F_1 = k_1 \Delta X_1 \hspace{2em} and \hspace{2em} F_2 = k_2 \Delta X_2\]
\(\Rightarrow\) F = \(k_1\) \(\Delta X_1\) + \(k_2\) \(\Delta X_2\) = ( \(k_1 + k_2\) ) \(\Delta X\)
Therefore, for spring in parallel:
\[\boxed{k_{eff} = k_1 + k_2 + ..... + k_N}\]
